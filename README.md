# VCliteWorkspace
Off market exchange
```
To get started
- clone repo
- pipenv install (To install dependencies from pipfile)
- python manage.py runserver 
- To startup front-end 'cd' to vc-app
- npm install (To install dependencies from package.json)
- npm run dev
```
